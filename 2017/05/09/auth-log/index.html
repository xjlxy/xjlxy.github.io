auth.log文件不存在

2017-05-09
#ubuntu #rsyslog #know errors

Error

/var/log/auth.log
阿X云部分主机没有这个文件

问题排查

首先确认rsyslog进程是否正常运行：

排查rsyslog配置：

可以看到也是默认的配置，并没有做任何改动。

md5sum对比两个服务器(正常，异常)的rsyslog.conf，md5值一致
有资料提到ssh配置文件的问题，比较后两个服务器配置文件也是一致的

在保证配置文件是一致的情况下，重启了rsyslog、ssh，但是/var/log/auth.log 依然没有

尝试手动创建文件

断开ssh重新登录
cat /var/log/auth.log
已经开始记录信息
大致可以断定问题在权限上

继续排查目录权限
最终定位：/var/log

可以看出，的确是权限导致的
